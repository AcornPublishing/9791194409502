## 소스 코드 다운로드 방법

다음 두 가지 방법 중 하나를 선택하세요.

- **ZIP 파일로 다운로드**: 페이지 상단의 `<> Code` 버튼을 클릭한 후 `Download ZIP`을 선택합니다.
- **Git으로 복제**: 터미널에서 `git clone` 명령어를 사용하여 저장소를 복제합니다.
